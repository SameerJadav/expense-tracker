# Fuzi

Modern example app built with Go in the backend and Next.js in the frontend with features like authentication, middleware, and API routes.

## Features

- Authentication
- Authorization
- Middleware

## Stack

**Backend:** Go
**Frontend:** Next.js
**Authentication:** Goth
**Asynchronous state management:** TanStack Query
