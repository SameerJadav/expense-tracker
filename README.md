# Expense Tracker

Modern example app built with Go in the backend and Vite + React in the frontend with features like authentication, middleware, and API routes.

## Features

- Authentication
- Authorization
- Middleware

## Stack

**Backend:** Go
**Frontend:** Vite + React
**Router:** TanStack Query
**Authentication:** Goth

## Tables

**Expenses**

| Column  | Type           |
| ------- | -------------- |
| id      | integer        |
| user_id | text           |
| title   | text           |
| amount  | numeric(12, 2) |
